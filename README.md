# aiagent-work-guid
작업 가이드 프로젝트


---

### ✅ 프롬프트 템플릿

```
아래는 내가 수행하려는 업무 목표이다. 이 문장을 기반으로,
1) 업무의 최종 산출물 정의
2) 추진 단계(상위 단계 → 하위 세부 단계)
3) 단계별 필요 자료/협업 대상
4) 단계별 예상 리스크 및 대응 방법
으로 정리해줘.

업무 목표: "<여기에 한 줄 목표 입력>"
```

---

### 사용 예시

**입력:**
업무 목표: "신규 서비스 기획안을 만든다"

**결과 (예시 출력):**

* 최종 산출물: 기획서 문서(PPT 15p 내외)
* 추진 단계:

  1. 시장/경쟁사 조사
  2. 타겟 고객 정의
  3. 핵심 가치 제안 정리
  4. 서비스 플로우 및 UI 개략 설계
  5. 기획서 작성 및 내부 공유
* 필요 자료/협업: 마케팅팀 인터뷰, 이전 기획 문서
* 리스크: 방향 모호 → 초기에 문제 정의를 문서화하여 공유
